# 🚀 CivicGuide AI

> Simplifying Government Services with AI

CivicGuide AI is an AI-powered web platform that helps citizens easily access government services such as passport applications, driving license guidance, scheme recommendations, and document verification.

Built for hackathons to solve real-world public service accessibility problems.

---

# 🌟 Features

## 🤖 AI Chat Assistant
Ask any government-related question like:

- How to apply for passport?
- Driving license process?
- PAN card documents?
- Aadhaar update process?

AI gives structured instant responses.

---

## 📑 Scheme Recommender 

Users enter:

- Age
- Income
- Gender
- State

And AI recommends eligible government schemes.

---

## 📷 OCR Verification 

Upload documents like:

- Aadhaar Card
- PAN Card
- Driving License

System verifies text using OCR.

# ⚙️ Installation & Setup
## Clone Repository
git clone https://github.com/yourusername/civic-guide.git
cd civic-guide

## Install Frontend Dependencies
cd client
npm install

## Start Frontend
npm run dev

## Install Backend Dependencies
cd server
npm install

## Create Environment File
 -> Inside server/ create: .env
 -> Add: OPENROUTER_API_KEY=your_api_key_here

## Start Backend
nodemon server.js
